# 🛍️ ShopHub

Este projeto é uma aplicação de e-commerce desenvolvida com React e Next.js, como parte das atividades práticas do curso.

## ✅ Entregas

### Entrega 1 – Crie sua Landing
- Criação dos componentes base: `NavBar`, `CartWidget` e `ItemListContainer`.
- Uso de props para exibir uma mensagem de boas-vindas.
- Estrutura inicial da homepage da loja.

### Entrega 2 – Navegue pelas Rotas
- Implementação da navegação entre páginas:
  - Página inicial
  - Lista de produtos por categoria
  - Página de detalhe do produto
- Uso de rotas dinâmicas e organização por pastas.

## 🚀 Tecnologias
- React
- Next.js (App Router)
- Tailwind CSS
- shadcn/ui
